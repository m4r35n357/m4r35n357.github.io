Particle

Object Hierarchy:

Description:

public class Particle : Object, IGenerator

Turns fixed parameters and other constraints into initial conditions data suitable as o to Models.Bh3d (passing some items directly through to the output data), alternatively takes the data and creates potential plots from it.
