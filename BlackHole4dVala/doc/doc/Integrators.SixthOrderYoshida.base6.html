base6

Description:

protected void base6 (double s)

Yoshida composition from 4th order to 6th order. Performs the following calls per iteration:

base4(s * y1)
base4(s * y2)
base4(s * y1)

where y1 = 1 / (2 - 2**(1/5)), and y2 = 1 - 2 * y1
