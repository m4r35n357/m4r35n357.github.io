Symplectic

Description:

public Symplectic (IModel model, double h, string label, string scheme)

This constructor produces instances from its label and scheme arguments according to the following tables:

label Integrator Order Description
"b1" firstOrder 1st Order, Symplectic, NOT Reversible
"b2" secondOrder 2nd Order, Symplectic, Reversible
"b4" fourthOrder 4th Order, Symplectic, Reversible
"b6" sixthOrder 6th Order, Symplectic, Reversible
"b8" eightthOrder 8th Order, Symplectic, Reversible

Parameters:

model

the model

h

the time step

label

the integrator order

scheme

the composition scheme
