getIntegrator

Description:

public ISymplectic getIntegrator (IModel model, double h, string type)

Static factory for producing subclass instances from its type argument according to the following table:

type Subclass Description
"b1" FirstOrder 1st Order, Symplectic
"b2" SecondOrder 2nd Order, Symplectic, Reversible
"sb4" FourthOrderSuzuki 4th Order, Symplectic, Reversible
"sb6" SixthOrderSuzuki 6th Order, Symplectic, Reversible
"sb8" EightthOrderSuzuki 8th Order, Symplectic, Reversible
"yb4" FourthOrderYoshida 4th Order, Symplectic, Reversible
"yb6" SixthOrderYoshida 6th Order, Symplectic, Reversible
"yb8" EightthOrderYoshida 8th Order, Symplectic, Reversible
"kl6" SixthOrderKahanLi9o6b 6th Order, Symplectic, Reversible
"kl8" EightthOrderKahanLi17o8b 8th Order, Symplectic, Reversible

Parameters:

h

the time step

type

the selected implementation

Returns:

concrete implementation of a symplectic integrator
