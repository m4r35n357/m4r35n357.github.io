getIntegrator

Description:

public ISymplectic getIntegrator (IModel model, double h, string type)

Static factory for producing subclass instances from its type argument according to the following table:

type Subclass Description
"sb1" FirstOrder 1st Order, Symplectic
"sb2" SecondOrder 2nd Order, Symplectic, Reversible
"sb4" FourthOrder 4th Order, Symplectic, Reversible
"sb6" SixthOrder 6th Order, Symplectic, Reversible
"sb8" EightthOrder 8th Order, Symplectic, Reversible

Parameters:

h

the time step

type

the selected implementation

Returns:

concrete implementation of a symplectic integrator
