Interfaces:

  • ISolver - External user interface for the physical model
  • IModel - Internal interface for the symplectic integrator to access and update the model
  • ISymplectic - Internal interface for a model to drive the symplectic integrator

Classes:

  • Bh3d - A model for generating geodesics in the Kerr- deSitter spacetime
  • Body - Description of a single Newtonian body using its coordinates, momenta and mass
  • NBody - Hamiltonian model for Newtonian N body orbits
  • Newton - Hamiltonian model for Newtonian central body orbit
  • Pendulum - Hamiltonian model for a simple pendulum
  • Oscillator - Hamiltonian model for a Harmonic Oscillator
  • Symplectic - Symplectic integrator abstract superclass, leaves integration method selection to subclasses
  • FirstOrder - First-order symplectic integrator concrete subclass. This integrator is NOT time symmetrical.
  • SecondOrder - Second-order symplectic integrator concrete subclass. This integrator is time symmetrical.
  • FourthOrder - Fourth-order symplectic integrator concrete subclass. This integrator is time symmetrical.
  • SixthOrder - Sixth-order symplectic integrator concrete subclass. This integrator is time symmetrical.
  • EightthOrder - Eightth-order symplectic integrator concrete subclass. This integrator is time symmetrical.
