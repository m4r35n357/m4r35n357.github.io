Namespaces:

  • Generators - Generators of initial conditions for simulating geodesics in the Kerr spacetime, and also of potential data

    • Classes:

    • Particle - Turns fixed parameters and other constraints into initial conditions data suitable as input to Simulations.BhSymp (passing some items directly through to the output data), alternatively takes the data and creates potential plots from it.
    • Light - Generates initial conditions and potentials for light
  • Simulations - Various simulations based on symplectic integration of separable Hamiltonian problems

    • Interfaces:

    • IGenerator - Internal interface for the parameter generators
    • ISolver - External user interface for the physical model
    • IModel - Internal interface for the symplectic integrator to access and update the model
    • ISymplectic - Internal interface for a model to drive the symplectic integrator

    Classes:

    • Symplectic - Symplectic integrator abstract superclass, leaves integration details to subclasses
    • EulerCromer - First-order symplectic integrator concrete subclass. This integrator is NOT time symmetrical.
    • StormerVerlet - Second-order symplectic integrator concrete subclass. This integrator is time symmetrical.
    • ForestRuth - Fourth-order symplectic integrator concrete subclass. This integrator is time symmetrical.
    • BhSymp - A model for generating geodesics in the Kerr-deSitter spacetime
    • Body - Description of a single Newtonian body using its coordinates, momenta and mass
    • NBody - Hamiltonian model for Newtonian N body orbits
    • Newton - Hamiltonian model for Newtonian central body orbit
