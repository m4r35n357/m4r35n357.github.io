main

Description:

public void main (string[] args)

Common entry point for all simulators based on symplectic integration, reads in JSON data from stdin and executes a program according to the following table:

program name key JSON variable class(es)) simulation
Simulate "Newton" Models.Newton Newtonian central body problem
Simulate "KerrDeSitter" Models.Bh3d Kerr-deSitter geodesics
Simulate "NBody" Models.NBody Models.Body Newtonian N body problem
GenParticle N/A Generators.Particle Generate initial conditions for Kerr-deSitter particle geodesics
GenLight N/A Generators.Light Generate initial conditions for Kerr-deSitter light geodesics

Parameters:

args

the program is hard linked, so args[0] is used as the program name
